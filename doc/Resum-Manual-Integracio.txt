
Dades importants:

(1) API Externa (Estadístiques): Accés Públic
      (1.a) Swagger : https://HOST:PORT/carpetaapi/externa/index.html  (Per exemple  https://www.caib.es/carpetaapi/externa/index.html)
      (1.b) API JAVA:

<!-- Carpeta - API Rest - Externa - Client -->
<dependency>
    <artifactId>carpeta-api-externa-client</artifactId>
    <groupId>es.caib.carpeta</groupId>
    <version>1.1</version>
</dependency>


(2) API Externa (Notificacions al Mòbil del Ciutadà):  Accés Restringit amb usuari/contrasenya. Requereix rol CAR_ADMIN o CAR_SUPER.
     (2.a) Swagger: https://HOST:PORT/carpetaapi/interna/index.html  (Per exemple  https://carpeta.fundaciobit.org/carpetaapi/interna/index.html)
     (2.b) API JAVA:

<!-- Carpeta - API Rest - Externa - Client -->
<dependency>
    <artifactId>carpeta-api-interna-client</artifactId>
    <groupId>es.caib.carpeta</groupId>
    <version>1.1</version>
</dependency>
